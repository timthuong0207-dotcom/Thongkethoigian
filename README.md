# Web Thống Kê Thời Gian

Trang web này cho phép nhập ngày sự kiện và tính toán thời gian từ lúc đó đến hiện tại theo năm, tháng, tuần, ngày, giờ, phút, giây.  
Được viết bằng HTML, CSS, JavaScript.  

## Cách sử dụng
1. Mở bằng trình duyệt.
2. Nhập ngày sự kiện.
3. Xem kết quả hiển thị realtime.
